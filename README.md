# Ledger

Command line app for ledger modelling in Java. 

### Requirements

    gradle > 7.0 
    java ~ 11

### Running test

    ./gradlew test

### Building jar

    ./gradlew jar
Find the jar in build/libs/ledger-co.jar

### How to use

    java -jar build/libs/ledger-co.jar <src/main/resources/transactions-1.txt>
